# Admin V2：江苏中考真题采集目录

本版本在 V1 题库管理后台基础上增加“真题采集”页面。

## 已加入

- 2022—2026 年；
- 江苏 13 个设区市；
- 数学、语文、英语、物理、化学；
- 共 325 个试卷采集任务；
- 来源链接、文件取得状态、解析状态、审核状态、发布状态；
- 批量标记进度和导出来源清单。

## 使用

1. 将 `admin/` 和 `data/` 上传到 GitHub 仓库根目录；
2. 访问 `/admin/`；
3. 点击“真题采集”；
4. 点击“载入目录”；
5. 打开来源，取得你有权使用的 PDF/Word/图片；
6. 将文件整理成 JSON/Excel 后通过“上传题库”导入。

## 重要说明

本目录只保存公开来源索引，不复制第三方完整试卷正文。部分网站需要登录、付费或仅提供图片。完整题目正文应来自官方公开授权来源，或由你本人合法取得并上传。
